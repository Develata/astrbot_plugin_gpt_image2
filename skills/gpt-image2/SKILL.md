# GPT Image 2 插件使用说明

当用户明确要求“画一张图”“生成图片”“改这张图”“把这张图变成某种风格”等图像生成/编辑任务时，可以使用本插件。

## 重要原则

- 只在用户明确要求生成或编辑图片时触发。
- 不要为了同一请求重复调用工具。
- 工具返回的是后台任务 `job_id`，不是最终图片。
- 最终图片会由插件后台直接发送到当前会话；LLM 不需要也不会看到图片结果。

## 文生图

使用 LLM Tool：

```text
gpt_image2_generate(prompt, size, quality)
```

推荐默认：

- `size`: `1536x1024`
- `quality`: `medium`

## 改图

如果当前消息附带图片，或引用消息里有图片，并且用户明确要求改图，使用：

```text
gpt_image2_edit(prompt, size, quality)
```

若没有可用参考图，不要调用改图工具；应要求用户附图或引用图片。

## 避免误触发

以下情况不要调用：

- 用户只是在讨论图片概念，没有要求实际生成。
- 用户问“怎么画/如何写 prompt”，但没有要求现在生成。
- 工具刚刚返回 job_id，用户没有提出新的独立生图需求。
